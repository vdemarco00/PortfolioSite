Lab Recording Helper

An EEG recording & realtime monitoring tool developed for use with Muse headbands. Check it out here.

This tool receives input from both the Muse EEG headband and other programs broadcasting stimulus event signals over LSL (Lab Streaming Layer), and records all the information at the correct times to a .csv file.

When an external program that displays stimuli is running at the same time of the recording, the markers are recorded alongside the EEG data and can be used to quickly determine when the stimulus start and end relative to the EEG data, which will help researchers find the relevant data sets they need quickly.
